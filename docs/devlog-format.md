# Dev Log Format

Write simple development logs while building.

Example:

2026-03-08

Today:

-   Fixed Supabase auth issue
-   Learned Next.js middleware
-   Generated migration with Claude Code

Guidelines:

-   Keep logs short
-   Focus on problems and solutions
-   List key discoveries

These logs will be transformed by AI into blog posts.

## Conversion Example

Dev Log:

-   Fixed Supabase auth issue

Blog Post:

Problem: Auth session not persisting.

Solution: Correct middleware session handling.

Lesson: Next.js middleware behaves differently from API routes.
