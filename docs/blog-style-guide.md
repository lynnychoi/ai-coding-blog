# Blog Writing Style Guide

The AI should always generate posts using the following structure.

# Title

## Problem

Explain what problem occurred.

## Solution

Explain how it was solved.

## Code Example

Include relevant code snippets.

## What I Learned

Explain the key takeaway.

## One-line Summary

Summarize the lesson in one sentence.

## Writing Style

-   Short paragraphs
-   Clear explanations
-   Developer friendly
-   Practical examples
-   No unnecessary fluff

Goal: Make posts readable and useful for other developers.
