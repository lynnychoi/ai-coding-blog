# Project Overview

## Goal

Create a blog that automatically documents what is learned while coding
with AI.

Instead of manually writing articles, the system converts development
logs into readable blog posts.

## Key Principles

1.  Development comes first
2.  Writing is automated
3.  Knowledge is archived

## Core Concept

Dev Log → AI Processing → Blog Post

## Blog Characteristics

-   Clean developer blog
-   Markdown based
-   Chronological posts
-   Tag system
-   Category system

## Audience

-   Developers learning AI coding
-   Builders documenting their process
-   People exploring AI-assisted development
