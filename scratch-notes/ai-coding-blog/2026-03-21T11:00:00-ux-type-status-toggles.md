# UX 개선: 타입/공개여부 드롭다운 → 토글 버튼

## 날짜
2026-03-21

## 변경 전
- 편집/생성 페이지에서 타입, 공개여부를 `<select>` 드롭다운으로 선택
- 모바일에서 불편, 줄줄이 나열

## 변경 후
- 드롭다운 → 탭 형태 토글 버튼
- **타입**: `dev` (보라색) / `writing` (회색)
- **공개여부**: `공개` (초록) / `비공개` (보라)
- 날짜 + 타입 + 공개여부 **한 줄**에 배치 (데스크탑/모바일 모두)
  - 데스크탑: 넓게 잘 보임
  - 모바일(390px): "비공개" 12px로 줄여서 한 줄 안에 들어옴

## 파일
- `app/cooking/edit/[slug]/page.tsx`
- `app/cooking/write/page.tsx`
