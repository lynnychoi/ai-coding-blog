# generation-log

글 생성/수정 이력. 재료 → 프롬프트 → 결과물 흐름을 추적한다.

## 파일명 규칙
`{slug}.md` — 글 슬러그 기준으로 파일 하나.
예: `2026-03-20-being-alone-with-baby.md`

## 파일 구조
최신 버전이 위에 온다.

```markdown
# {slug}

## v2 — 2026-03-20T16:00 (최신)
- **재료**: (notes 원문)
- **추가 프롬프트**: (없으면 없음)
- **태그**: tag1, tag2
- **타입**: dev / writing

---

## v1 — 2026-03-20T14:00
- **재료**: ...
- **추가 프롬프트**: ...
- **태그**: ...
- **타입**: ...
```
